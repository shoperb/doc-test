# Getting Started

Shoperb Theme Editor is a ruby gem created to make building Shoperb themes easier. Only on macOS and Linux based operating systems are supported today. Windows is currently not tested by our team but may be working.

If you run into any trouble then get in touch with us on Shoperb Themes channel on Gitter and we’ll try to help you out.

# Installation

RUBY VERSION SUPPORT

We do not currently support Ruby 2.5.

First we need to have Ruby installed on our system. You can check if you already have Ruby on your system by executing ruby -v in console.

If you don’t have Ruby then installing Ruby using RVM is pretty straightforward.

# Project Setup

BEFORE YOU START

Basic understanding of how to navigate in file structure, edit files etc in macOS or Linux console is required.

Examples assume that theme folder is located at ~/Projects/Shoperb-Themes/Canvas

# Gemfile

You should create a Gemfile in project folder using your favourite editor.
Following content should be copy-pasted into Gemfile.

source "https://rubygems.org"
gem "shoperb-theme-editor", github: "shoperb/theme-editor"
gem "artisans", "~> 2.0",  github: "shoperb/artisans"

# Install gems with Bundler

bundle install

# Confirm that Shoperb Theme Editor was successfully installed

shoperb --help

If everything went well you can now start creating your Shoperb Theme. In the unfortunate case where shoperb --help does not output anything or gives an error message, get in touch with us on Shoperb Themes channel and we’ll help you out.
